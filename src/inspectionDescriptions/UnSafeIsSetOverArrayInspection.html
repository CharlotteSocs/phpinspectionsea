Analyses 'isset(...)' function usages.

As analysis performed, inspection suggests to use 'null !== ...', 'array_key_exists(..., ...)' instead. In cases when isset applied to array access expression, inspection checks if concatenation is used for an index.

Couple words regarding array_key_exists: it will return true even if array item is null, so it's more about verifying data structures.

Working out array_key_exists refactoring recommendation will cost additional development and legacy code reverse engineering, so it shall be planned accordingly.
