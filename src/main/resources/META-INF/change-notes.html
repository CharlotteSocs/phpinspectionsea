New features release

- Declaration access can be weaker: resolved a false (child classes exists)
- Unused function result: resolved a false-positive (fluent interfaces)
- 'yield from' can be used: resolved a false-positive (yield from in a loop)

- Badly organized exception handling: new patterns
- Low performing directory operations: implemented
- One-time use variables: new patterns
- Overriding deprecated methods: new patterns
- Slow array function used in loop: new patterns
- Suspicious binary operations: new patterns
- 'explode(...)' misused: new patterns
